## Installation

**To be able to run app locally:**

- install and run redis locally
- rename .example.env to .development.env
- specify REDIS_PORT in .development.env if redis is running not on the 6379 port

```bash
$ npm install
```

## Running the app

```bash
# development

$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
